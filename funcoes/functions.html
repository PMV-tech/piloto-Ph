BarberFlow Functions

Documentação das funções JavaScript disponíveis no sistema

Estas funções estão disponíveis globalmente através do objeto window.BarberFlow
formatDate(date, format)
Formata uma data para exibição. Aceita objetos Date, strings ou timestamps.
date
Data a ser formatada (Date, string ou timestamp)
format (opcional)
Formato da data (padrão: 'dd/MM/yyyy')
// Exemplos de uso:
BarberFlow.formatDate(new Date());
// Resultado: "15/03/2024"

BarberFlow.formatDate('2024-03-15', 'dd/MM/yyyy HH:mm');
// Resultado: "15/03/2024 14:30"
formatCurrency(value)
Formata um valor numérico como moeda brasileira (Real).
value
Valor numérico a ser formatado
// Exemplos de uso:
BarberFlow.formatCurrency(150.75);
// Resultado: "R$ 150,75"

BarberFlow.formatCurrency('2850');
// Resultado: "R$ 2.850,00"
formatPhone(value)
Formata um número de telefone para o padrão brasileiro.
value
Número de telefone a ser formatado
// Exemplos de uso:
BarberFlow.formatPhone('11999998888');
// Resultado: "(11) 99999-8888"

BarberFlow.formatPhone('1133334444');
// Resultado: "(11) 3333-4444"
isValidEmail(email)
Valida se uma string é um endereço de e-mail válido.
email
Endereço de e-mail a ser validado
// Exemplos de uso:
BarberFlow.isValidEmail('usuario@email.com');
// Resultado: true

BarberFlow.isValidEmail('usuario@email');
// Resultado: false
showMessage(message, type, duration)
Exibe uma mensagem toast na tela. Tipos disponíveis: 'success', 'error', 'warning', 'info'.
message
Texto da mensagem a ser exibida
type (opcional)
Tipo da mensagem (padrão: 'info')
duration (opcional)
Duração em milissegundos (padrão: 5000)
// Exemplos de uso:
BarberFlow.showMessage('Operação concluída!', 'success');

BarberFlow.showMessage('Erro ao salvar dados.', 'error', 3000);
confirmAction(message, confirmText, cancelText)
Exibe um modal de confirmação personalizado. Retorna uma Promise que resolve com true (confirmado) ou false (cancelado).
message
Mensagem de confirmação
confirmText (opcional)
Texto do botão confirmar (padrão: 'Confirmar')
cancelText (opcional)
Texto do botão cancelar (padrão: 'Cancelar')
// Exemplo de uso:
async function deleteItem() {
  const confirmed = await BarberFlow.confirmAction('Deseja excluir este item?');
  if (confirmed) {
    // Executar exclusão
  }
}
showLoading() / hideLoading()
Exibe e esconde um overlay de loading na tela.
// Exemplo de uso:
BarberFlow.showLoading();
// Executar operação assíncrona...
BarberFlow.hideLoading();
exportToCsv(data, filename)
Exporta um array de objetos para um arquivo CSV.
data
Array de objetos a serem exportados
filename (opcional)
Nome do arquivo (padrão: 'export.csv')
// Exemplo de uso:
const data = [
  { nome: 'João', email: 'joao@email.com' },
  { nome: 'Maria', email: 'maria@email.com' }
];
BarberFlow.exportToCsv(data, 'clientes.csv');
